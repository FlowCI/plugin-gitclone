### 描述
一个 Git Clone 的插件，适用于 FlowCi

### 输入参数
>- PLUGIN_API `Api的服务地址` `必填`
>- PLUGIN_GIT_URL `Git的克隆地址` `选填`
>- PLUGIN_GIT_BRANCH `Git的分支` `选填`
>- PLUGIN_GIT_WORKSPACE `Git的克隆的目录` `选填`
